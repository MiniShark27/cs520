# How to run TicTacToe

### To Compile:
run the following:
```bash
cd tictactoe
ant compile
```

### To Run:
run the following: (If already compiled, don't need to run `ant compile`)
```bash
cd tictactoe
ant compile
java -cp bin RowGameApp
```

### To Test:
run the following:
```bash
cd tictactoe
ant test
```
